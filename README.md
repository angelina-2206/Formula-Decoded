**Formula Decoded** is an immersive, cinematic web experience designed to visually dissect the engineering, aerodynamics, and hybrid power systems of modern Formula One cars. With a focus on the upcoming **2026 FIA regulations**, the platform transforms complex motorsport concepts into an interactive visual narrative.

---

## 🌟 Key Features

### 🛠️ Interactive 3D Engineering
Explore a high-fidelity 3D model of a Formula One car. Zoom into components, rotate the chassis, and interact with hotspots to learn about the Front Wing, Halo, Sidepods, and more.
*   **Technologies:** Three.js, React Three Fiber, GLB Compression.

### 🌬️ Active Aerodynamics Simulation
Visualize the future of F1 aerodynamics. Toggle between **Corner Mode** (high downforce) and **Straight Mode** (low drag) to see real-time wing adjustments and particle-based airflow behavior.

### ⚡ Hybrid Energy Visualization
Experience the power of the 2026 Power Unit. Simulate energy deployment through three distinct modes:
*   **Overtake Mode:** Maximum electric boost.
*   **Boost Mode:** Optimized hybrid output.
*   **Recharge Mode:** Kinetic energy recovery during braking.

### 📂 Exploded Engineering View
Break the car apart into its core assemblies. A scroll-driven "Exploded View" allows you to inspect the drivetrain, chassis, and aero components in isolation.

### 🏁 Beyond the Machine
*   **Driver Grid:** Animated profiles of the current F1 grid with team-specific theming.
*   **Race Calendar:** Interactive global map showing upcoming Grand Prix locations.
*   **Evolution Timeline:** A scroll-driven journey through the history of F1, from the 1950s to the 2026 era.

## 🛠️ Tech Stack

- **Frontend:** [Next.js](https://nextjs.org/), [React](https://reactjs.org/)
- **3D Engine:** [Three.js](https://threejs.org/), [React Three Fiber](https://docs.pmnd.rs/react-three-fiber)
- **Animations:** [GSAP](https://greensock.com/gsap/), [Framer Motion](https://www.framer.com/motion/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Optimization:** [Draco Compression](https://google.github.io/draco/), [Lighthouse](https://developers.google.com/web/tools/lighthouse)

## 🚀 Getting Started

### Prerequisites
- Node.js 18.x or higher
- npm / pnpm / yarn

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/angelina-2206/Formula-Decoded.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) to see the result.
